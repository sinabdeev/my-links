# My Links

Хранилище избранных ссылок с организацией по вкладкам.

Не используется сервер. Простой HTML-файл с JavaScript.

## Структура проекта

```
my-links/
├── index.html              # Основная HTML-страница
├── app.js                  # Главный скрипт, инициализация приложения
├── styles.css              # Стили (темная тема Dracula)
├── data/
│   ├── tabs.js             # Массив названий табов
│   ├── table-1.js          # Данные первой вкладки
│   ├── table-2.js          # Данные второй вкладки
│   ├── table-3.js          # Данные третьей вкладки
│   ├── table-4.js          # Данные четвертой вкладки
│   └── table-5.js          # Данные пятой вкладки
└── js/
    ├── activateTab.js      # Логика переключения табов
    ├── loadTableData.js    # Ленивая загрузка данных таблиц
    ├── renderTable.js      # Рендеринг таблиц
    ├── cellNavigation.js   # Клавиатурная навигация по ячейкам
    └── tooltip.js          # Тултип с задержкой показа
```

## Формат данных

Каждая строка — массив из 5 ячеек с полями `text` и `url`:

```js
const table1 = [
    [
        {text: "Название", url: "https://example.com"},
        {text: "Описание", url: "https://example.com/desc"},
        {text: "Категория", url: "https://example.com/cat"},
        {text: "Статус", url: "https://example.com/status"},
        {text: "Дата", url: "https://example.com/date"}
    ]
];
```

## Запуск

Откройте [`index.html`](index.html) в браузере.

## Особенности

- **Темная цветовая схема Dracula** — современная палитра с акцентными цветами
- Динамическое переключение табов с сохранением в `localStorage`
- Поддержка URL-хэшей (например, `#tab0`)
- Ленивая загрузка таблиц по требованию
- Адаптивный дизайн
- Каждая ячейка таблицы — кликабельная ссылка
- **Hover-эффекты** — при наведении на ячейку меняется фон и цвет текста
- **Тултип с задержкой** — при наведении на ячейку URL показывается через 1 секунду; если курсор убран раньше — тултип не появляется
- **Клавиатурная навигация по вкладкам**: `Shift + →` — следующая вкладка, `Shift + ←` — предыдущая вкладка
- **Клавиатурная навигация по ячейкам**: стрелки `←` `→` `↑` `↓` — перемещение по ячейкам таблицы (пустые ячейки с отсутствующей ссылкой пропускаются, при достижении края — остановка), `Enter` — открыть ссылку активной ячейки в новой вкладке
