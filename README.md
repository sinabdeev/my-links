# My Links

Хранилище избранных ссылок с организацией по вкладкам.

Не используется сервер. Простой HTML-файл с JavaScript.

## Структура проекта

| Файл | Описание |
|------|----------|
| [`index.html`](index.html) | Основная HTML-страница с табами и стилями |
| [`tabs.js`](tabs.js) | Массив названий табов |
| [`table-1.js`](table-1.js) | Данные первой вкладки |
| [`table-2.js`](table-2.js) | Данные второй вкладки |
| [`table-3.js`](table-3.js) | Данные третьей вкладки |

## Формат данных

Каждая строка — массив из 5 ячеек с полями `text` и `url`:

```js
const table1 = [
    [
        {text: "Название", url: "https://example.com"},
        {text: "Описание", url: "https://example.com/desc"},
        {text: "Категория", url: "https://example.com/cat"},
        {text: "Статус", url: "https://example.com/status"},
        {text: "Дата", url: "https://example.com/date"}
    ]
];
```

## Запуск

Откройте [`index.html`](index.html) в браузере.

## Особенности

- Динамическое переключение табов с сохранением в `localStorage`
- Поддержка URL-хэшей (например, `#tab0`)
- Ленивая загрузка таблиц по требованию
- Адаптивный дизайн
- Каждая ячейка таблицы — кликабельная ссылка
- **Клавиатурная навигация по вкладкам**: `Shift + →` — следующая вкладка, `Shift + ←` — предыдущая вкладка
- **Клавиатурная навигация по ячейкам**: стрелки `←` `→` `↑` `↓` — перемещение по ячейкам таблицы (пустые ячейки с отсутствующей ссылкой пропускаются, при достижении края — остановка), `Enter` — открыть ссылку активной ячейки в новой вкладке
