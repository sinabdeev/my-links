# План исправления бага выделения ячеек при навигации стрелками

## Описание проблемы
При навигации по ячейкам таблицы нажатием стрелок (←→↑↓) навигация работает (ячейки меняются), но визуальное выделение активной ячейки (фон и цвет текста) не появляется.

## Анализ кода

### Файлы, участвующие в проблеме:

1. **[`js/cellNavigation.js`](js/cellNavigation.js:57-63)** — функция `setActive()` добавляет класс `cell-active` к ячейке
2. **[`styles.css`](styles.css:114-117)** — стили `.cell-active` определяют `outline` и `background`
3. **[`styles.css`](styles.css:82)** — базовый стиль ячеек `.data-table td` устанавливает `background: var(--dracula-selection)`
4. **[`styles.css`](styles.css:104-111)** — hover-эффект `.data-table td:hover` меняет фон и цвет текста

### Корневая причина бага:

**Стили `.cell-active` не совпадают с hover-эффектом**:
- Hover-эффект ([`styles.css:104-111`](styles.css:104)): 
  - Фон: `var(--dracula-purple)` (фиолетовый)
  - Цвет текста: `var(--dracula-bg)` (тёмный)
  - Цвет ссылки: `var(--dracula-bg)` (тёмный)
- Активная ячейка ([`styles.css:114-117`](styles.css:114)):
  - Фон: `var(--dracula-selection)` (совпадает с обычным фоном ячеек)
  - Только `outline: 2px solid var(--dracula-cyan)` — малозаметный индикатор

## Решение

Изменить стили `.cell-active` на идентичные hover-эффекту:

### Изменения в [`styles.css`](styles.css:114-117):

**Текущий код:**
```css
/* Выделение активной ячейки при клавиатурной навигации */
.data-table td.cell-active {
    outline: 2px solid var(--dracula-cyan);
    background: var(--dracula-selection);
}
```

**Новый код:**
```css
/* Выделение активной ячейки при клавиатурной навигации */
.data-table td.cell-active {
    background: var(--dracula-purple);
    color: var(--dracula-bg);
}

.data-table td.cell-active a {
    color: var(--dracula-bg);
}
```

### Обоснование изменений:

1. **Фон `var(--dracula-purple)`** — идентичен hover-эффекту, обеспечивает контрастное выделение
2. **Цвет текста `var(--dracula-bg)`** — тёмный текст на светлом фоне для читаемости
3. **Цвет ссылки `var(--dracula-bg)`** — ссылка тоже становится тёмной, как при hover
4. **Убран `outline`** — не нужен при таком контрастном выделении

## Результат

✅ При нажатии стрелок активная ячейка визуально выделяется:
- Фон ячейки меняется на `#bd93f9` (Dracula purple)
- Цвет текста меняется на `#282a36` (Dracula bg)
- Цвет ссылки внутри активной ячейки — `#282a36` (Dracula bg)
- При перемещении на другую ячейку предыдущая теряет выделение
- Hover-эффект продолжает работать для неактивных ячеек
- Поведение идентично наведению мыши на ячейку
